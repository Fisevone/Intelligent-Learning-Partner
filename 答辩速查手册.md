## 💡 高频问题速答

### Q1: 为什么用Kotlin?
**答**: Google官方推荐、语法简洁、空安全、协程支持、与Java兼容

### Q2: MVVM优势?
**答**: 分离关注点、可测试性强、生命周期感知、数据驱动、易维护

### Q3: AI如何实现?
**答**: Retrofit调用智谱AI API → 构建Prompt → 协程异步请求 → 解析JSON → 展示结果

### Q4: 如何保证数据安全?
**答**: 加密存储、HTTPS传输、权限控制、匿名化处理、用户同意

### Q5: 学习推荐原理?
**答**: 收集用户数据 → 提取特征 → 计算相似度 → AI辅助分析 → 排序推荐 → 反馈优化

### Q6: 协程优势?
**答**: 轻量级、结构化并发、异常处理、取消支持、代码可读性高

### Q7: Room优势?
**答**: 编译时验证、减少样板代码、LiveData/Flow支持、类型安全、迁移管理

### Q8: 如何处理网络失败?
**答**: 重试机制(3次) → 备用模型 → 降级方案 → 错误提示 → 离线模式

### Q9: 角色权限如何控制?
**答**: userType字段区分 → 权限检查 → 数据过滤 → 界面分离 → 关联关系

### Q10: 未来规划?
**答**: 语音识别、视频通话、AR/VR、iOS版、Web版、商业化

## 🎓 专业术语解释

### MVVM
Model-View-ViewModel，一种架构模式，将UI、业务逻辑、数据分离

### Room
Android官方ORM库，简化SQLite数据库操作

### Coroutine (协程)
Kotlin的轻量级并发框架，简化异步编程

### LiveData
Android架构组件，可观察的数据持有类，生命周期感知

### Flow
Kotlin的异步数据流，支持响应式编程

### Retrofit
类型安全的HTTP客户端，简化网络请求

### ViewBinding
Android视图绑定，替代findViewById，类型安全

### DAO
Data Access Object，数据访问对象，封装数据库操作

### Repository
仓库模式，统一数据访问接口，隔离数据源

### Prompt
提示词，用于引导AI生成特定内容的输入文本
